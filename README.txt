Block Burst — Complete Merged Version

Includes:
- Full 100-level block puzzle game
- No-creep drag system
- Startup artwork screen
- Looping background music
- Music on/off button
- Saved level progress

How to play:
1. Unzip this folder.
2. Open index.html in Chrome.
3. Press Start Game.
4. Music begins after the first click/tap because browsers block autoplay.

Files:
- index.html
- assets/startup.png
- assets/music.mp3

For GitHub Pages:
Upload index.html and the assets folder to a public repository, then enable Pages.
