Block Burst — Complete Audio Fixed Version

Includes:
- Full 100-level block puzzle game
- No-creep drag system
- Startup artwork screen
- Looping background music
- Music on/off button
- Click sound when a block is placed
- Fail sound when a block cannot be placed
- Fanfare when a level is completed
- Saved level progress

Audio fix:
Browsers block sounds until a user interaction. This version unlocks all audio from the Start Game button, then uses cloned audio nodes for reliable overlapping sound effects.

How to play:
1. Unzip this folder.
2. Open index.html in Chrome.
3. Press Start Game.
4. Music and sound effects should now work.
